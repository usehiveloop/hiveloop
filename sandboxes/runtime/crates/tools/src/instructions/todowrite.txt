Track multi-step work as a structured todo list. Use proactively for any task with 3+ distinct steps; skip for trivial single-step tasks.

## States
- `pending`, `in_progress` (limit ONE at a time), `completed`, `cancelled`

## Priority
- `high`, `medium`, `low`

## Semantics
Every call REPLACES the entire list. Always send all items (completed, in-progress, pending). Update status immediately as work progresses; mark completed as soon as a task finishes (don't batch).

When you start a task, set its status to `in_progress`. Finish current tasks before starting new ones.
