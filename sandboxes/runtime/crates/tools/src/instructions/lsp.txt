Code intelligence via LSP. Use before editing typed code (find symbols, definitions, callers); use after editing to check diagnostics.

## Operations
- `goToDefinition` / `findReferences` / `goToImplementation` (line, character)
- `hover` (line, character) — types + docs
- `documentSymbol` (no position) — symbols in file
- `workspaceSymbol` (query) — search project-wide, up to 10 results
- `prepareCallHierarchy` / `incomingCalls` / `outgoingCalls` (line, character)
- `diagnostics` (no position) — errors/warnings for the file

## Args
- `operation`, `file_path` (required); `line`, `character` (1-based) for position ops; `query` for `workspaceSymbol`.

Prefer `goToDefinition` / `findReferences` over RipGrep when navigating typed code.
