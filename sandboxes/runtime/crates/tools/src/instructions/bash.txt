Execute a shell command. Each call runs in a fresh shell — env vars, `cd`, aliases do NOT carry across calls. Every invocation starts in the workspace directory.

Use for terminal operations (git, npm, composer, artisan, docker, tests). Do NOT use for file I/O — use Read, Write, Edit, Glob, RipGrep instead.

## Args
- `command` (required)
- `workdir`: absolute path to cd into for this call only
- `timeout`: ms, default 120000
- `description`: 5–10 words for logs
- `background: true`: spawn async, returns task_id, notification on completion

## Output
Output >~2 KB is summarized (head + tail) with full version spilled to a temp file path. Use RipGrep on that path to find specific content — do NOT re-run the command.

## Parallelism
Independent commands: multiple bash calls in one turn run in parallel. Dependent commands: chain with `&&` in one call.
