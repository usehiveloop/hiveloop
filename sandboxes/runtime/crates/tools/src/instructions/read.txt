Read a file from the local filesystem. The `file_path` must be an absolute path.

Defaults: returns up to 2000 lines from the start. For files >2000 lines, pass `limit` explicitly (use `offset`+`limit` for ranged reads, or RipGrep to locate specific content). Files >10MB are refused — use RipGrep, or spawn `self_agent` if you need to read multiple slices independently.

- `offset`: 1-indexed start line.
- `limit`: max lines to read.
- Contents are prefixed with `<line>: <content>`. Lines >2000 chars are truncated.
- Call in parallel when reading multiple files.
- Avoid tiny repeated slices (30-line chunks). If you need more context, read a larger window.
- Reads image files as base64 attachments.
