Search the web for information. Returns structured results with title, URL, and snippet.

- Provides up-to-date information for current events and recent data
- Results include knowledge graph data (when available) and organic search results
- Use this tool for accessing information beyond your knowledge cutoff

Usage notes:
  - The current year is {{year}}. Include the current year in time-sensitive queries to get the most relevant results
  - Example: if looking for "latest Rust release", search for "Rust release {{year}}" rather than just "Rust release"
  - Keep queries concise and specific for best results
  - Use web_fetch to retrieve full content from URLs found in search results
