Recursive grep with smart defaults. Searches the workspace by content.

- `pattern`: regex (Rust regex flavor).
- `path`: absolute path to search; defaults to cwd.
- `--type <lang>`: scope to one language (e.g. `--type rust`, `--type php`).
- `-n`: show line numbers (recommended; use file:line in output).
- `-C 3` / `-A 3` / `-B 3`: context lines.
- `-l`: list files only.
- `-i`: case-insensitive.

Skips `vendor/`, `node_modules/`, `target/`, `.git/`, generated dirs by default. Always prefer scoped queries — `rip_grep -n --type rust 'fn foo' crates/` over a full-tree scan.
