Fast file pattern matching tool that works with any codebase size.

- Supports glob patterns like "**/*.rs", "src/**/*.ts", "*.{js,jsx}"
- Returns matching file paths sorted by modification time (newest first)
- Results are truncated at 1000 matches
- Respects .gitignore rules automatically

Usage notes:
  - Use this tool when you need to find files by name or extension
  - The path parameter defaults to the current working directory if omitted
  - Use Glob to find files by name, Grep to search file contents, LS to list a directory
  - Call multiple Glob searches in parallel when you need to find different file types simultaneously
  - For open-ended searches that may require multiple rounds of globbing and grepping, describe the intent rather than issuing many small queries
