Read the conversation journal. Returns all journal entries including agent notes and checkpoint summaries from previous context chains.

## When to Use This Tool

- When you need to recall a decision or discovery you previously recorded
- When the user asks about what happened earlier in the conversation
- When you need to check if something was already journaled before writing a duplicate
- After a context chain handoff, to review what the journal contains

## What You'll Get

The journal contains two types of entries:

1. **agent_note** — Entries you wrote via journal_write (decisions, discoveries, preferences)
2. **checkpoint** — Automatic summaries extracted during context chain handoffs

Each entry includes the chain index it was written during, so you can trace the conversation's history across context resets.
