Convert HTML content to markdown or plain text without making any HTTP requests. Processes HTML you already have.

## When to Use
- Convert raw HTML from a previous tool call into readable markdown
- Clean up HTML content before summarizing or analyzing it
- Transform multiple HTML snippets in a single call
- Process HTML stored in files or received from APIs

## Parameters
- `data` (required): Array of HTML content items to transform. Each item has:
  - `html` (required): The HTML content string
  - `url` (optional): Source URL for resolving relative links
- `return_format`: Output format — "markdown" (default), "text" (plain text), "html2text" (simplified text extraction)

## Tips
- Include the source `url` when HTML contains relative links — they'll be resolved to absolute URLs
- Use "markdown" format for content you need to read, summarize, or pass to another tool
- Process multiple items in a single call rather than making separate calls for each
