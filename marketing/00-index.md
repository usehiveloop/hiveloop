## Quick Reference

**Positioning**: Secure proxy layer for LLM API keys. Store, mint, proxy — so customer credentials never touch your code.

**Primary headline**: "Your customers' LLM keys deserve a vault, not a database column."

**Primary persona**: Platform engineers building AI products with BYOK or sandbox access.

**GTM model**: Product-led growth (free tier → Pro → Enterprise).

**Key differentiator**: Purpose-built for LLM credential management — not a generic gateway, not just a secret manager. Full lifecycle: store → mint scoped tokens → proxy → revoke instantly.

**Pricing**: Free ($0) → Pro ($49/mo) → Enterprise (custom).
